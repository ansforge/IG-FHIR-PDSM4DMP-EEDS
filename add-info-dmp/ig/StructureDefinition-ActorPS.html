Logical Model: ActorPS (LM)

Official URL: https://interop.esante.gouv.fr/ig/fhir/pdsm4dmp/StructureDefinition/ActorPS Version: 0.1.0
Draft as of 2025-05-06 Computable Name: ActorPS

Cet attribut représente un acteur PS.

Usage:

ActorPS (LM)
Instances of this logical model can be the target of a Reference
XCN1[x] 1..1 PSIdNat Identifiant de l'acteur
Slice: Unordered, Open by type:$this
Exemple Professionnel avec un identifiant national RPPS (préfixe 8): 801234567890

XCN1[x] 1..1 PSIdNat, MatriculeINS, SNR, IdentifiantSysteme Identifiant de l'acteur
Required Pattern: D
XCN1[x] 1..1 PSIdNat, MatriculeINS, SNR, IdentifiantSysteme Identifiant de l'acteur
Required Pattern: IDNPS
XCN2 1..1 string Nom d'exercice du professionnel, nom du patient, nom du système.
XCN3 1..1 string Prénom usuel de la personne (par défaut le premier prénom), nom du modèle pour les dispositifs ou dénomination pour les autres systèmes.
XCN9 1..1 Base Autorité d’affectation
composant2 1..1 string Valeur de Universal ID
Required Pattern: 1.2.250.1.71.4.2.1
composant3 1..1 string Valeur de Universal ID type (ID)
XCN10 1..1 code Type de nom : Valeur en fonction de l’auteur : D, pour les personnes physiques, • U, pour les systèmes.
XCN13 1..1 code Type d’identifiant
XCN2 Nom d'exercice du professionnel
XCN3 Prénom usuel de la personne
