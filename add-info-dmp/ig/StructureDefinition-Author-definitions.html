0. Author
Définition

Modèle logique d'un auteur. Représente les personnes physiques et/ou les systèmes (dispositifs, automates, services numériques référencés…) auteurs d’un document ou d'un lot de soummission. L’auteur peut être :

  • Un professionnel (personne physique) via son logiciel de professionnel,
  • Le patient/usager (personne physique) via Mon espace Santé,
  • Un système de structure (dispositif, automate, appareil connecté…),
  • Un SNR (Service Numérique Référencé),
ShortAuthor (LM)
